## 文件

文件操作的步骤：打开、读写、关闭

### 读取文本文件

```python
(nsd1908) [root@room8pc16 day03]# cp /etc/passwd /tmp/
>>> f = open('/tmp/passwd')  # 默认以r方式打开
>>> data = f.read()   # 默认读取全部内容
>>> print(data)
>>> data = f.read()   # 继续向后读取
>>> data
''
>>> f.close()

>>> f = open('/tmp/passwd')
>>> f.read(4)  # 指定一次最多读取的字节数
'root'
>>> f.read(4)
':x:0'
>>> f.readline()  # 继续向后读取一行
':0:root:/root:/bin/bash\n'
>>> f.readlines()  # 继续向后读取，将所有的行读取到列表中
>>> f.close()

```
