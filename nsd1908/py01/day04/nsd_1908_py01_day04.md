## shutil模块

```python
>>> import shutil
>>> f1 = open('/bin/touch', 'rb')
>>> f2 = open('/tmp/tch', 'wb')
>>> shutil.copyfileobj(f1, f2)
>>> f1.close()
>>> f2.close()

>>> shutil.copy('/etc/hosts', '/tmp/')  # cp /etc/hosts /tmp/
'/tmp/hosts'

>>> shutil.copy2('/etc/hosts', '/tmp/zhuji')  # cp -p /etc/hosts 
                                                      /tmp/zhuji
'/tmp/zhuji'

```
