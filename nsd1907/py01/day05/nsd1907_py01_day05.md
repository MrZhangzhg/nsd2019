## 列表

- 属于容器、可变、序列类型

```python
>>> import random
# 创建具有5个随机数字的列表
>>> alist = [random.randint(1, 100) for i in range(5)]
>>> alist
[85, 21, 95, 85, 30]
# 赋值
>>> alist[-1] = 21
>>> alist
[85, 21, 95, 85, 21]
>>> alist[1:3] = [1, 3, 5, 7, 9]
>>> alist
[85, 1, 3, 5, 7, 9, 85, 21]

```
