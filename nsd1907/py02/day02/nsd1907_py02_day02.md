## 函数

### 参数

- 以key=val形式存在的参数，称作关键字参数
- 以args形式存在的参数，称作位置参数

```python
>>> def func1(name, age):
...   print('%s is %s years old' % (name, age))
... 
>>> func1()   # 错误，参数个数不够
>>> func1('tom', 20, 200)   # 错误，参数个数太多
>>> func1('tom', 20)   # OK
tom is 20 years old
>>> func1(20, 'tom')   # 语法正确，语义不对
20 is tom years old
>>> func1(age=20, name='tom')  # OK
>>> func1(age=20, 'tom')  # 语法错误，关键字参数必须在位置参数后
>>> func1(20, name='tom')  # 错误，因为name得到了多个值
>>> func1('tom', age=20)   # OK

```
